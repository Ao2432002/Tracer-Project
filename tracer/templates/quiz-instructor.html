Timer instructions:

  • When only a start and end date is set, the quiz is opened for this period only
  • When only a start date and time is set, the quiz is opened for all students. When this time passes, the quiz is closed
  • When a start and end date and time are set, the quiz is opened for all students and each student is given this specific time individually.
  • When nothing is specified, the quiz is opened and closed manually by the instructor
